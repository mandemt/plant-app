# folder backend:

``npm install``
## set up local database
.env file:
``` 
DB_PASSWORD='password here'
DB_NAME='database name here'
DB_USER='username here' 
DB_PORT=port nr
```


## folder plant-app

``npm install``

## start application

* in folder /plant-app/backend:
  - `npm start`
* in folder /plant-app/plant-app:
  * `npm run dev`
